Sintaxa liniei de comanda este urmatoarea:

java  Operatori  i  j  k  >  info.htm

unde i si j trebuie sa fie doua numere intregi cuprinse intre -9223372036854775808 si 9223372036854775807 
iar numarul k trebuie sa fie 0 sau 1. Valoarea k = 0 va determina afisarea operatiilor fara depasiri de 
domeniu iar k = 1 va determina afisarea operatiilor cu depasiri de domeniu.
